# Security Policy

## Supported versions

Security fixes are provided for the latest released version.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's private vulnerability reporting feature at **Security → Report a vulnerability** in this repository. If that option is unavailable, contact `mario@electricarts.de` with the subject “Local TablePro security report”.

Include the affected version, reproduction steps, impact, and any suggested mitigation. You should receive an acknowledgement within seven days. Please allow reasonable time for a fix before public disclosure.
