# Support

Before requesting help, install the latest release, restart Local and the affected site, and verify that TablePro opens independently.

Use [GitHub Discussions](https://github.com/electricarts/local-tablepro/discussions) for usage questions and [GitHub Issues](https://github.com/electricarts/local-tablepro/issues/new/choose) for reproducible defects or feature requests. Include Local, macOS, TablePro, MySQL, and add-on versions. Remove site names, filesystem paths, passwords, and other sensitive information from screenshots and logs.

This community project is maintained on a best-effort basis and is not official support from Local or TablePro.
